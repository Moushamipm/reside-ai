# Local Development Guide

This guide will help you run the PropAssist application locally with all dependencies stored inside the project folder.

## Prerequisites

Before you begin, ensure you have the following installed:

- **DFX (Internet Computer SDK)**: Version 0.15.0 or higher
- **Node.js**: Version 18.0.0 or higher
- **pnpm**: Version 8.0.0 or higher (recommended) or npm
